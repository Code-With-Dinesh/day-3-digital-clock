# ⏰ Digital Clock

A simple digital clock built using HTML, CSS, and JavaScript.

## 🚀 Features

- Live Digital Clock
- AM/PM Format
- Current Date Display
- Dark/Light Mode Toggle
- Modern UI Design

## 🛠️ Tech Used

- HTML
- CSS
- JavaScript
